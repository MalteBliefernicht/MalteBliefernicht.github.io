Moin!

Dies ist eine interaktive Arbeitsprobe, die ich kurzfristig für mein Vorstellungsgespräch bei der i-gelb GmbH zusammengebastelt habe.

Ich habe im Juni 2020 angefangen, das Programmieren zu lernen. Angefangen habe ich mit Python. Nach einem halben Jahr kam Java dazu. Außerdem habe ich mich an der Entwicklung von Android-Apps, webbasiertem Programmieren mit dem Python Framework "Django" und dem klassischen LAMP-Stack und ein bisschen C ausprobiert.

Mein Ziel beim Lernen war es zu prüfen, ob eine Ausbildung zum Fachinformatiker für mich in Frage kommen könnte. Ich habe schnell festgestellt, dass ich nicht nur durchaus ein Talent für das Programmieren habe, sondern dass es mir auch sehr viel Spaß macht.

Ich hoste diese Seite auf GitHub Pages. Leider sind dadurch nur statische Seiten möglich. Eigentlich hätte ich gerne auch PHP und Datenbanken eingesetzt.

Unter "Demos" können drei Javascript-Demonstrationen ausprobiert werden. Unter "Projekte" finden sich Einblicke in einige meiner älteren Projekte, die ich nicht problemlos in Javascript übersetzen konnte.
